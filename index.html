Volunteer Database Management Tool

Browser-local volunteer roster, attendance, tags, deduplication, review, and export tool.

Privacy notice: This tool stores volunteer data only in this browser's localStorage. Do not use shared devices for real personal data. Exported files contain personal data and should be protected.

Import Data

Choose the template type, then upload an Excel workbook or paste copied spreadsheet cells. Both methods require exact headers in exact order.

Upload .xlsx

Drop .xlsx here or click to select a file.

Maximum file size: 5 MB. First row must be the exact template header row.

Paste spreadsheet cells

Copy the header row and data rows from Excel or Google Sheets, then paste below. Maximum pasted text size: 1 MB.

Detected rows preview

Invalid rows are shown but will not be imported.

Merge Review

Nothing is written to the database until Confirm Import is clicked. Use pre-merge edits before confirming.

XLSX Export
