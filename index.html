MakLom

Browser-local volunteer roster, attendance event log, tags, deduplication, review, analytics, and export tool.

How MakLom works

MakLom runs in your browser and stores its working database locally on this device. It does not send volunteer data to a server.

Use Upload to import a roster or attendance event log template. MakLom validates headers, previews rows, and stages clean records, conflicts, and suspected duplicates for review before saving.

Use Merge Review to add tags, batch-edit staged rows, resolve conflicts, and confirm the import. Use Central Database to search, filter, edit profiles, and view attendance event log history.

Use Dashboard to review volunteer counts, recruitment by year, deployment by year, and programme participation.

Use Export to download full, redacted, merge-log, or JSON backup files. Exported files may contain personal data and should be handled securely.
